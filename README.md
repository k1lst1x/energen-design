# Energen Design Preview

Интерактивный дизайн-прототип Energen, развернутый из Figma Make export.

Оригинальный Figma Make файл:
https://www.figma.com/make/4KBWkStBTl6fxS17QfEBxk/AI-Assistant-Web-App-Design

## Назначение

Этот репозиторий нужен как дизайн-песочница для согласования экранов, тем, навигации и поведения цифрового помощника Energen. Финальный `energen-web` можно будет реализовывать отдельно на Vue, опираясь на утвержденные экраны отсюда.

## Запуск

```bash
npm install
npm run dev
```

Локальный URL по умолчанию:

```text
http://localhost:5173/
```

Фиксированный локальный порт для просмотра:

```bash
npm run dev:local
```

```text
http://127.0.0.1:5177/
```

## Проверка сборки

```bash
npm run build
```

## Темы

В прототипе есть темная и светлая темы. Переключатель находится в верхней навигации.

Также тему можно принудительно открыть через query-параметр:

```text
http://127.0.0.1:5177/?theme=light#/
http://127.0.0.1:5177/?theme=dark#/
```

## Основные маршруты

- `/` - главный экран с быстрыми действиями.
- `/chat` - текстовый чат с Energen.
- `/voice` - голосовой режим.
- `/room` - карточка аудитории.
- `/employee` - карточка сотрудника.
- `/appointment` - запись на прием.
- `/events` - мероприятия.
- `/clubs` - студенческие клубы.
- `/payments` - оплата.
- `/directions` - как добраться.

## Структура

- `src/app/pages` - страницы прототипа.
- `src/app/components` - общие компоненты, включая layout и avatar.
- `src/app/context/LanguageContext.tsx` - тексты RU/KZ/EN.
- `src/app/context/ThemeContext.tsx` - переключение dark/light темы.
- `src/styles` - глобальные стили, темы и шрифты.
- `src/assets` - логотипы Energo University.
- `BRAND_NOTES.md` - заметки по логотипам и цветам.
- `ATTRIBUTIONS.md` - атрибуция исходных UI-компонентов и изображений.

## Перед публикацией

В репозиторий не добавляются `node_modules`, `dist` и локальные скриншоты проверок. Для публикации на GitHub достаточно создать пустой репозиторий и запушить текущую git-историю.
